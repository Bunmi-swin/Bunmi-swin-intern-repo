## Top ten shortcuts

### Navigation

- Ctrl+F: Find a word
- Ctrl+G: Go to a specific line within the file.

### Editing

- Alt+Click:  Add an additional cursor on the line that is manually clicked
- Ctrl+Alt+Down or Ctrl+Alt+Up: Add additional cursors directly above or below the current one
- Ctrl+D: Add another cursor and select the next instance of a word
- Ctrl+L:  Select everything within a line
- Ctrl+S: Save file.

### Debugging

- F8/Shift+F8: Go to the next error or warning within a file.-

### Terminal Commands

- Ctrl+C: Copy selection
- Ctrl+V: Paste selection into terminal/file.

## Reflection

### Which shortcuts do you think will be most useful in your daily work?

- I already often use Ctrl+C, Ctrl+S and Ctrl+V to save some time instead of typing. Ctrl+G \& Ctrl+D seem useful in the same way, especially if I need to edit multiple variables at the same time or a particular file contains a large number of lines.

### How can keyboard shortcuts improve productivity?

- They streamline time spent on repetitive or common actions, allowing for operations to be done efficiently or in a less roundabout method. By doing so, more time and effort is spent on work that requires problem-solving, research, or needs a set time to complete itself.
